# StudentHub Portal

StudentHub is a simple student portal developed using HTML.

## Aim

To design and develop a StudentHub portal for managing student academic activities.

## Problem Scope

Students need a single platform to access academic information such as assignments, attendance, notices and study materials.

## User Roles

- Student
- Faculty
- Admin

## Modules

- Login
- Dashboard
- Student Profile
- Assignments
- Attendance
- Study Material
- Notices
- Logout

## Technologies Used

- HTML
- Visual Studio Code
- Git
- GitHub
